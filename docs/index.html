FakePromise

Fake Promise class, a promise like class to support caching functionality.

new FakePromise()
Example
const cache = new PromiseCache("mypath", (res, rej) => {
  res(50);
});
const inst = new FakePromise(cache);
inst.then(shouldBe(50)).then(done.bind(null, void 0));

then

Attaches callbacks for the resolution and/or rejection of the Promise.

then
Returns
FakePromise<(TResult1 | TResult2)>: A new Fake Promise for the completion of which ever callback is executed.
Example
const cache = new PromiseCache("", (res, rej) => {
   res(50);
}, false);
const inst = new FakePromise(cache);
inst.then(console.log); // => 50

catch

Attaches a callback for only the rejection of the Promise.

catch
Returns
FakePromise<(T | TResult)>: A new Fake Promise for the completion of the callback.
Example
const cache = new PromiseCache("", (res, rej) => {
  rej("My Error 2");
}, false);
const inst = new FakePromise(cache);
inst.catch((err) => {
  console.warn(err);
});

finally

Attaches a callback for which will be called once all the main operations are completed.

finally
Returns
FakePromise<T>: A Fake Promise for the completion of the callback.
Example
const cache = new PromiseCache("", (res, rej) => {
  rej("My Error 2");
}, false);
const inst = new FakePromise(cache);
inst.finally(() => {
  // do cleanup..
});

PromiseCache

Promise Cache class, caches a given promise to a given path.

new PromiseCache()
Example
const cache = new PromiseCache("mypath", (res, rej) => {
  res(50);
});
cache.then(console.log); // => 50
